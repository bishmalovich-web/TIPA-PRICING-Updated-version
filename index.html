Film Pricing App

Excel-based calculator from Film Pricing sheet, including margin, input mode, length input, EUR and USD sell price.

Friendly pricing calculator

Choose film, width, thickness, density, weight or length, hourly rate, shipping destination and margin. The app follows the same calculation logic as the Excel calculator: Sell Price = Total Cost/Kg × (1 + Margin), and USD = Sell Price × exchange rate.

GitHub Pages — step by step

  1. Create or open your GitHub repository, for example tipa-pricing.
  2. Upload these files from the ZIP to the repository root: index.html, tipa-pricing-qr.png, and optional README.md.
  3. Go to Settings → Pages.
  4. Set Source to Deploy from a branch.
  5. Choose Branch: main and Folder: / root, then press Save.
  6. Open your GitHub Pages link. If your repo is tipa-pricing, use your Pages URL for that repo.
  7. Open the page on your computer, press Refresh QR, then scan the QR with your phone.

Important: GitHub Pages is static. This HTML is generated from the Excel values and formulas. If you later change the Excel prices/formulas, generate and upload a new index.html so the website matches the updated Excel.
